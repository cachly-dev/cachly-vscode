# Connect your Brain

Your Brain is a persistent memory that lives outside the editor — so your AI
assistant remembers lessons, fixes and context across sessions, restarts and
machines.

**One click** starts a free trial (no credit card) or lets you paste an
existing `cky_live_…` key. When the **status bar** at the bottom turns green
("Brain: N lessons"), you're connected.

> Tip: stuck on red "Brain: OFFLINE"? Run **Cachly: Run Brain Doctor** from the
> Command Palette — it tells you exactly what's wrong and how to fix it.
