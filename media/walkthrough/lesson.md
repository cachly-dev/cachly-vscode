# Save your first lesson

A **lesson** is anything worth remembering: a fix, a gotcha, a convention, a
command that worked. Save one and your Brain keeps it forever.

Run **Cachly: Save Lesson**, describe what you learned, and it's stored. Next
time you (or a teammate sharing the same Brain) hit the same situation, it
surfaces automatically.

> The extension also learns *ambiently*: when a compiler error disappears after
> an edit, it can offer to save that fix as a lesson for you.
